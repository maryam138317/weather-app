☁️ Simple React Weather App

This is a compact web application built with React to fetch and display current weather conditions. It focuses on integrating with a weather API and showcasing fundamental React development skills.

<br />

✨ Features
<ul>
    <li>Location-Based Search: Get real-time weather data for any city you search for.</li>
    <li>API Integration: Successfully handles asynchronous data fetching from an external API.
</li>
    <li>Responsive Design: Works well across desktop and mobile devices.
</li>
</ul>
